========================================
PATTERN #2 : Two Pointers (Opposite Ends)
========================================

Recognition Keywords:
- Reverse
- Palindrome
- Sorted array
- Pair from beginning and end
- Mirror comparison
- Reverse in-place

When NOT to Use:
- Filtering arrays
- Longest substring
- Prefix sums
- Binary Search

Main Idea:
Start one pointer from the left
and another from the right.

Move them toward each other.

Visualization:

H E L L O

L         R

↓

Swap

↓

Move inward

Pointers / Variables:
left
right

Algorithm:
1. left = 0
2. right = n-1
3. While left < right:
      Swap
      left++
      right--

Loop Invariant:
Everything outside
[left ... right]
is already in its correct position.

Time Complexity:
O(n)

Space Complexity:
O(1)

Common Mistakes:
- Using left <= right
- Forgetting pointer updates
- Wrong stopping condition
- Using extra array unnecessarily

Common LeetCode Problems:
✓ Reverse String
✓ Valid Palindrome
✓ Reverse Vowels
✓ Two Sum II
✓ Squares of a Sorted Array
✓ Container With Most Water

Interview Tips:
Whenever the problem compares
the beginning and end of a sequence,
think:
Two Pointers.
